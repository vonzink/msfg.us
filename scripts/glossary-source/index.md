## Term link pattern
Each term links to: `/resources/mortgage-glossary?term=<slugified-term>`
Example: `1003 Form` → `/resources/mortgage-glossary?term=1003-form`

## Sections
Terms are grouped under letter headings (#, A, B … Z). Within each
section, terms are listed alphabetically with the term as a heading
and the definition as body text. See the per-letter content files.
